# NEZ-TOOL

Многофункциональный инструмент для сетевого анализа, тестирования и автоматизации, написанный на Java.

## Описание

NEZ-TOOL - это комплексный набор утилит для работы с сетевыми протоколами, Discord API, веб-сервисами и системным анализом. Инструмент предоставляет удобный консольный интерфейс для выполнения различных задач тестирования и автоматизации.

## Основные возможности

### Сетевые инструменты
- **Сканер портов** - быстрое сканирование открытых портов с поддержкой многопоточности
- **HTTP стресс-тестирование** - отправка множественных HTTP/HTTPS запросов для нагрузочного тестирования
- **IP анализатор** - получение детальной информации об IP адресах и доменах
- **Прокси-сервер** - встроенный HTTP прокси с возможностью перехвата трафика

### Discord интеграция
- **Webhook спам** - массовая отправка сообщений через Discord webhooks
- **Бот управление** - полнофункциональный Discord бот с административными возможностями
- **Серверное управление** - создание/удаление каналов, ролей, управление участниками
- **Автоматизация** - массовые операции с серверами Discord

### Дополнительные функции
- **Прокси менеджер** - управление прокси-соединениями
- **Многопоточность** - оптимизированная работа с большими объемами данных
- **Гибкая конфигурация** - настройка параметров через файлы конфигурации

## Детальное описание модулей

### Function1 - Discord Webhook Спам
- Отправка сообщений через Discord webhooks
- Поддержка загрузки webhooks из файла
- Многопоточная отправка с настраиваемой задержкой
- Обработка rate-limiting и повторные попытки
- Валидация URL и обработка ошибок

### Function2 - Сканер портов
- Быстрое сканирование популярных портов
- Настраиваемые диапазоны портов
- Определение сервисов по портам
- Многопоточное сканирование (до 50 потоков)
- Детальная статистика сканирования

### Function3 - HTTP стресс-тестирование
- Поддержка GET, POST, HEAD методов
- Настраиваемое количество запросов и потоков
- Кастомные User-Agent и POST данные
- Мониторинг кодов ответов
- Настраиваемые таймауты и задержки

### Function4 - IP анализатор
- Получение геолокации IP адресов
- Информация о провайдере и организации
- Определение часового пояса
- Reverse DNS lookup
- Интеграция с ip-api.com и ipinfo.io

### Function5 - Discord бот
- Полное управление Discord серверами
- Массовые операции с каналами и ролями
- Управление участниками (бан, кик, мут)
- Спам функции и автоматизация
- Интуитивное меню управления

### Proxy система
- **ProxyServer** - HTTP прокси сервер
- **ProxyClient** - клиент для подключения к прокси
- **ProxyWorker** - обработчик прокси запросов
- **ProxyManager** - управление прокси соединениями

## Требования

- **Java 17** или выше
- **Maven 3.6+**
- Интернет соединение для API запросов
- Discord токены (для Discord функций)

## Установка

1. **Клонирование репозитория:**
```bash
git clone https://github.com/typefaceQ/nez-tool.git
cd nez-tool
```

2. **Сборка проекта:**
```bash
mvn clean compile
```

3. **Создание jar файла:**
```bash
mvn package
```

## Использование

### Запуск основного приложения
```bash
java -jar target/crasher-1.0-SNAPSHOT.jar
```

### Запуск прокси-сервера
```bash
# Linux/macOS
./run-proxy.sh

# Windows
run-proxy-windows.bat
```

### Автозапуск worker (Windows)
```batch
worker-autostart.bat
```

## Конфигурация

### Настройка токенов
Создайте папку `tokens/` в корне проекта:

```
tokens/
├── discord_token.txt          # Discord бот токен
├── config.properties          # Основная конфигурация
└── webhooks.txt              # Список Discord webhooks
```

### Пример config.properties
```properties
discord.token=YOUR_DISCORD_BOT_TOKEN
proxy.host=localhost
proxy.port=8080
threads.max=100
timeout.connection=5000
```
