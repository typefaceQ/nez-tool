# NEZ-TOOL

Мульти-инструмент для различных задач, написанный на Java.

## Описание

NEZ-TOOL - это многофункциональный инструмент, который включает в себя различные утилиты для работы с сетью, Discord ботами и другими задачами.

## Возможности

- Работа с Discord API
- Сканирование портов
- Прокси-сервер функциональность
- Веб-хуки
- И многое другое

## Требования

- Java 17 или выше
- Maven 3.6+

## Установка

1. Клонируйте репозиторий:
```bash
git clone https://github.com/yourusername/NEZ-TOOL.git
cd NEZ-TOOL
```

2. Соберите проект:
```bash
mvn clean compile
```

## Использование

### Linux/macOS
```bash
./run-proxy.sh
```

### Windows
```batch
run-proxy-windows.bat
```

## Конфигурация

Создайте папку `tokens/` и добавьте необходимые токены и конфигурационные файлы.

## Структура проекта

```
NEZ-TOOL/
├── src/main/java/CrAsHeR/     # Основной код
├── tokens/                    # Токены и конфигурация (не включены в git)
├── scripts/                   # Скрипты запуска
└── pom.xml                   # Maven конфигурация
```
