# Быстрая настройка для Windows 11

## Шаг 1: Установка Java

1. Скачайте Java 17: https://adoptium.net/
2. Установите (Next → Next → Install)
3. Проверьте: откройте CMD и введите `java -version`

## Шаг 2: Подготовка файлов

Создайте папку, например: `C:\ProxyWorker\`

Скопируйте туда:
- `crasher-1.0-SNAPSHOT-jar-with-dependencies.jar`
- `worker-autostart.bat`

## Шаг 3: Настройка

Откройте `worker-autostart.bat` в Блокноте и измените:
```bat
set RASPBERRY_IP=192.168.1.100
```
На IP вашего Raspberry Pi

## Шаг 4: Запуск

Двойной клик на `worker-autostart.bat`

Должно появиться:
```
╔════════════════════════════════════╗
║   PROXY WORKER - ПК ПОМОЩНИК       ║
╚════════════════════════════════════╝
[*] Worker ID: abc12345
[*] Подключение к: http://192.168.1.100:8080
```

## Шаг 5: Автозапуск (опционально)

### Вариант 1: Папка автозагрузки
1. `Win + R` → введите `shell:startup` → Enter
2. Создайте ярлык `worker-autostart.bat` в этой папке

### Вариант 2: Планировщик заданий
1. `Win + R` → введите `taskschd.msc` → Enter
2. Действия → Создать простую задачу
3. Имя: `Proxy Worker`
4. Триггер: `При входе в систему`
5. Действие: `Запустить программу`
6. Программа: `C:\ProxyWorker\worker-autostart.bat`
7. Готово

## Проверка работы

Воркер должен каждые 2 секунды проверять команды.
Когда придет команда, увидите:
```
[+] Получена команда: 1|token|guildid
[*] Спам через вебхуки...
[✓] Результат отправлен
```

## Остановка

Просто закройте окно или нажмите `Ctrl + C`

## Проблемы?

**"Java не найдена"**
- Переустановите Java
- Добавьте в PATH: `C:\Program Files\Eclipse Adoptium\jdk-17\bin`

**"JAR файл не найден"**
- Проверьте, что файл в той же папке что и .bat
- Проверьте название файла

**"Не подключается к серверу"**
- Проверьте IP Raspberry Pi
- Проверьте, что сервер запущен
- Проверьте firewall/антивирус
